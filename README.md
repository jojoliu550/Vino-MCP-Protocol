# 🍷 Vino-MCP: The AI Agent Wine Protocol
**Global Standard for Agent-Driven Wine Sourcing & Fulfillment.**
*(由意大利 JOJO IMPORT SRL 发起的全球首个“AI 智能体红酒直连采购协议”)*

---

### 🌟 什么是 Vino-MCP？
在 2026 年，人类已经不再需要亲自打开网页挑选商品。Vino-MCP 是一套基于最新 Model Context Protocol (MCP) 构建的开源接口标准。

它旨在让 Claude、ChatGPT、Cursor 等各种形态的 AI 智能体（Agent），能够绕过传统的电商平台，直接、结构化、秒速地读取意大利顶级红酒的库存、风味数据，并替人类雇主完成精准匹配与下单。

**我们不是在做一家网店，我们在为 AI 时代建立基础设施。**

---

### 🔥 为什么需要这个协议？
传统红酒选购充满摩擦：产区复杂、年份繁多、搭配困难。
有了 Vino-MCP：

* **AI 侍酒师无缝接入：** 你的 AI 管家可以在你规划晚餐菜单的同时，通过本协议瞬间检索出最适合搭配牛排的 Barolo（巴罗洛）。
* **零 UI 交易（M2M Commerce）：** 去除所有网页弹窗和繁琐的购物车，机器与机器直接对话，实现极致极简的交易链路。

---

### 🏗️ 核心功能 (Core Features)
* `get_wine_catalog`: AI 瞬间获取实时库存、产区、单宁/酸度维度数据。
* `sommelier_match`: 提供食物或场景（如：商务宴请、海鲜晚餐），系统自动返回最优的三款意版红酒方案。
* `agent_checkout`: (Beta 测试中) 通过加密 Token，AI 直接生成数字支付凭证，对接后端物理仓储发货。
